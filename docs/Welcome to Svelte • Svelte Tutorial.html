Svelte 튜토리얼에 오신 것을 환영합니다! 이 튜토리얼에서는 모든 규모의 웹 애플리케이션을 작은 설치 공간에서 고성능으로 쉽게 구축하는 데 필요한 모든 것을 알려드립니다.

API 문서와 예제를 참조하거나, 로컬 컴퓨터에서 해킹을 시작하고 싶지 않다면 npm init svelte 에서 프로젝트를 만들 수도 있습니다.

Svelte란 무엇인가요?

Svelte는 웹 애플리케이션을 구축하기 위한 도구입니다. 다른 사용자 인터페이스 프레임워크와 마찬가지로 마크업, 스타일 및 동작을 결합한 컴포넌트로 선언적으로 앱을 빌드할 수 있습니다.

이러한 컴포넌트는 작고 효율적인 자바스크립트 모듈로 컴파일되어 기존 UI 프레임워크와 관련된 오버헤드를 제거합니다.

이 튜토리얼에서 다루는 SvelteKit과 같은 애플리케이션 프레임워크를 사용하여 전체 앱을 Svelte로 빌드하거나 기존 코드베이스에 점진적으로 추가할 수 있습니다. 또한 컴포넌트를 어디서나 작동하는 독립형 패키지로 제공할 수도 있습니다.

이 튜토리얼을 사용하는 방법

Svelte를 이해하려면 HTML, CSS 및 JavaScript에 대한 기본적인 지식이 있어야 합니다.

이 튜토리얼은 크게 네 부분으로 나뉩니다:

  • 기본 Svelte (현재 위치)
  • 고급 Svelte
  • 기본 스벨트 키트
  • 고급 스벨트 키트

각 섹션에서는 기능을 설명하기 위해 고안된 연습 문제를 제시합니다. 이후 연습은 이전 연습에서 얻은 지식을 기반으로 하므로 처음부터 끝까지 진행하는 것이 좋습니다. 필요한 경우 위 메뉴를 통해 탐색할 수 있습니다.
