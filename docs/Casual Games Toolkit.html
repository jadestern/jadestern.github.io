Flutter 캐주얼 게임 툴킷은 모바일 플랫폼에서 게임 개발을 가속화할 수 있도록 새로운 리소스와 기존 리소스를 한데 모아 제공합니다.

이 페이지에서는 사용 가능한 리소스를 어디에서 찾을 수 있는지 간략하게 설명합니다.

왜 게임에 Flutter를 사용해야 할까요?

Flutter 프레임워크는 데스크톱, 모바일 기기, 웹 등 6가지 대상 플랫폼을 위한 고성능 앱을 만들 수 있습니다.

크로스 플랫폼 개발, 성능, 오픈 소스 라이선스 등 Flutter가 제공하는 이점을 고려할 때 Flutter는 게임에 적합한 선택입니다.

캐주얼 게임은 턴제 게임과 실시간 게임의 두 가지 범주로 나뉩니다. 두 가지 유형의 게임에 모두 익숙할 수도 있지만, 이런 식으로 생각하지 않았을 수도 있습니다.

턴제 게임은 간단한 규칙과 게임 플레이로 대중 시장을 겨냥한 게임으로, 보드 게임, 카드 게임, 퍼즐, 전략 게임 등이 여기에 포함됩니다. 이러한 게임은 카드를 탭하거나 숫자나 문자를 입력하는 등 간단한 사용자 입력에 반응하며, Flutter에 적합합니다.

실시간 게임은 일련의 동작에 실시간으로 반응해야 하는 게임을 포함합니다. 무한 달리기 게임, 레이싱 게임 등이 이에 해당합니다. 충돌 감지, 카메라 뷰, 게임 루프 등과 같은 고급 기능을 갖춘 게임을 만들고자 할 수 있습니다. 이러한 유형의 게임에는 Flutter를 사용하여 구축된Flame 게임 엔진과 같은 오픈 소스 게임 엔진을 사용할 수 있습니다.

캐주얼 게임 툴킷은 다음과 같은 무료 리소스를 제공합니다.

  • 캐주얼 게임을 빌드하기 위한 시작점을 제공하는 세 가지 새로운 게임 템플릿이 포함된 저장소.

    1. 기본 게임 템플릿에대한 기본 사항이 포함되어 있습니다:

      • 메인 메뉴
      • 탐색
      • 설정
      • 레벨 선택
      • 플레이어 진행 상황
      • 플레이 세션 관리
      • 사운드
      • 테마

    2. 기본 템플릿에 모든 것을 더한 카드 게임 템플릿입니다:

      • 드래그 앤 드롭
      • 게임 상태 관리
      • 멀티플레이어 통합 후크

    3. 오픈 소스 게임 엔진인 Flame과 제휴하여 만든 무한 러너 템플릿입니다. 구현합니다:

      • FlameGame 기본 템플릿
      • 플레이어 스티어링
      • 충돌 감지
      • 시차 효과
      • 스폰
      • 다양한 시각 효과

    4. 끝없는 러너 템플릿을 기반으로 제작된 샘플 게임, SuperDash. iOS, Android 또는 웹에서 게임을 플레이하고, 오픈 소스 코드 리포지토리를 확인하거나,6주 만에 게임을 만든 방법을 읽어볼 수 있습니다.

  • 필요한 서비스를 통합하기 위한 개발자 가이드.
  • 플레임 디스코드 채널 링크. 디스코드 계정이 있는 경우 이 직접 링크를 사용하세요.

포함된 게임 템플릿과 쿡북 레시피는 개발을 가속화하는 데 도움이 됩니다. provider, google_mobile_ads,in_app_purchase, audioplayers, crashlytics, games_services 와 같은 특정 패키지가 포함되어 있습니다. 다른 패키지를 선호하는 경우 코드를 변경하여 해당 패키지를 사용할 수 있습니다.

Flutter 팀은 수익 창출이 향후 고려 대상이 될 수 있음을 이해합니다. 광고 및 인앱 구매를 위한 요리책 레시피가 추가되었습니다.

게임 페이지에 설명된 대로 클라우드, 파이어베이스, 광고와 같은 Google 서비스를 게임에 통합하면 최대 900달러의 혜택을 받을 수 있습니다.

시작하기

준비되셨나요? 이제 시작하세요:

  1. 아직 설치하지 않았다면 Flutter를 설치합니다.
  2. 게임 리포지토리를 복제합니다.

  3. 만들고자 하는 첫 번째 게임 유형에 대한 README 파일을 검토합니다.

    • 기본 게임
    • 카드 게임
    • 러너 게임
  4. 디스코드에서 플레임 커뮤니티에 가입합니다(이미 디스코드 계정이 있는 경우 직접 링크 사용).

  5. 코드랩과 요리책 레시피를 검토합니다.

    • 클라우드 파이어스토어로 멀티플레이어 게임을 만들어 보세요.
    • 리더보드와 도전 과제를 통해 게임의 몰입도를 높이세요.
    • 게임 내 광고와 인앱 구매로 게임에서 수익을 창출하세요.
    • Firebase 인증을 통해 게임에 사용자 인증 흐름을 추가하세요.
    • Firebase 크래시리틱스로 게임 내 크래시 및 오류에 대한 분석을 수집하세요.
  6. 필요에 따라 애드몹, 파이어베이스, 클라우드에 계정을 설정합니다.
  7. 게임을 제작하세요!
  8. 구글 플레이와 애플 스토어에 모두 배포하세요.

게임 예시

Google I/O 2022를 위해 Flutter 팀과 Very Good Ventures는 새로운 게임을 만들었습니다.

  • VGV는 Flame 엔진을 사용하여 I/O 핀볼 게임을 만들었습니다. 이 게임에 대해 알아보려면 미디엄에서 Flutter와 Firebase로 구동되는 I/O 핀볼을 확인하고 브라우저에서 게임을 플레이해 보세요.

  • Flutter 팀은 가상 CCG인 I/O Flip을 만들었습니다. I/O Flip에 대해 자세히 알아보려면 Google 개발자 블로그에서 ' 어떻게 만들어졌나: 제너레이티브 AI로 고전적인 카드 게임에 색다른 재미를 더한 I/O FLIP'을 확인하고 브라우저에서 게임을 플레이해 보세요.
