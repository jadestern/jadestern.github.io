CSS를 어디에 배치하든 언어에 대한 숙달이 필요합니다. CSS를 더 잘하면 더 효과적인 프론트엔드 개발자가 될 수 있습니다.

이 튜토리얼에서는 CSS에서 가장 흥미로운 최신 개발 기능 중 하나인 사용자 정의 속성을 활용하는 방법을 살펴봅니다: 일명 사용자 정의 속성입니다. React 앱에서 이를 사용하여 워크플로를 개선하고 꽤 멋진 작업을 수행하는 방법을 살펴볼 것입니다.

왜 그럴까요?

React 개발자는 CSS에서 변수가 필요하지 않다고 생각할 수 있습니다. 자바스크립트 엔진 전체를 마음대로 사용할 수 있으니까요!

React 앱에서 CSS 변수로 전환해야 하는 이유는 두 가지입니다:

  1. 인체공학적으로 좋습니다.

  2. 새로운 가능성을 열어줍니다! CSS 변수를 사용하면 JS에서는 불가능한 일을 할 수 있습니다.

어떻게 사용하는지 살펴보고 어떤 문이 열리는지 알아봅시다!

CSS 변수는 font-size 또는 color 과 같이 기본적으로 상속 가능합니다. 요소에 변수를 정의하면 해당 요소의 모든 하위 요소에서 사용할 수 있습니다.

CSS 변수에 대한 몇 가지 간단한 사실입니다:
  • 사용자 정의 프로퍼티는 대시 두 개로 시작해야 합니다. 이것이 기존 CSS 프로퍼티와 구별되는 점입니다.
  • 색상과 픽셀뿐만 아니라 모든 유형의 값을 저장할 수 있습니다.
  • CSS 변수가 정의되지 않은 경우 기본값을 지정할 수 있습니다. var(--primary-color, pink) 은 필요한 경우 pink 으로 되돌아갑니다.

웹 개발자로서 우리는 세 가지 주요 기술을 사용합니다: HTML, CSS, JS입니다. 효과적으로 작업하려면 이 세 가지 기술에 모두 익숙해야 합니다.
저는 HTML과 JS를 꽤 잘 이해하지만 CSS에 어려움을 겪는 개발자를 많이 알고 있습니다. 저도 몇 년 동안 이 그룹에 속해 있었습니다!
CSS는 놀라울 정도로 복잡한 언어입니다. 기본은 빠르고 (비교적) 쉽게 배울 수 있지만 마스터하기는 엄청나게 어려운 언어입니다.
CSS를 작성할 때 우리는 빙산의 일각만 보는 것입니다. 매우 암시적인 언어이기 때문입니다. 온갖 종류의 비밀 메커니즘이 작동하여 놀랍고 이해할 수 없는 방식으로 CSS 속성의 효과를 조정합니다.
CSS에는 오류 메시지나 콘솔 로그가 없습니다. CSS를 작성하면 예상한 대로 결과가 나오거나 그렇지 않을 수도 있습니다. 일이 잘못되면 무력감을 느낄 수 있습니다. 결국 임의의 CSS 프로퍼티를 벽에 던져놓고 그 중 하나가 원하는 대로 작동하기를 바라게 됩니다.
제가 돕고 싶습니다. 저는 최근에 자바스크립트 개발자를 위한 CSS 강좌를 출시했습니다. 이 강좌는 React와 같은 JS 프레임워크를 사용하는 분들을 위해 특별히 제작되었습니다. 이 강좌는 CSS가 실제로 어떻게 작동하는지 알려주고, 언어에 대한 직관력을 키울 수 있도록 멘탈 모델을 구축하는 방법을 알려줍니다.
지금까지 수강한 다른 강좌와는 다릅니다. 150개가 넘는 동영상이 있지만 동영상 강좌는 아닙니다. 대화형, 다양한 방식의 자기 주도적 모험입니다. 다양한 연습 문제, 실제 세계에서 영감을 얻은 프로젝트, 심지어 몇 가지 미니 게임도 포함되어 있습니다. 이전에 사용했던 것과는 다른 맞춤형 플랫폼을 사용합니다.
CSS에 어려움을 겪고 있다면 꼭 한번 확인해 보시기 바랍니다. 특히 HTML과 JS에 이미 익숙하다면 CSS에 자신감을 얻는 것은 게임 판도를 바꿀 것입니다. 이 삼위일체를 완성하면 웹 애플리케이션을 개발하는 것이 훨씬 더 쉬워지고 진정한 즐거움을 느낄 수 있습니다.
