서론

이 글은 React 외부에서 프론트엔드 개발을 공부하는 데 초점을 맞춘 시리즈의 일부입니다. 시리즈 공부의 시작은 Svelte입니다.

이 글은 이론적으로나 실무적으로 Svelte에 대한 중요한 사항을 요약한 참고 자료가 될 것입니다.

저는 React 개발자로서 윈도우 기법, 컴포넌트와 경로의 지연 로딩, 데이터와 이미지의 비동기 로딩, 로컬 리소스의 동적 로딩과 같이 효과가 입증된 기능에 해당하는 것을 찾으려고 노력해왔습니다.

제가 연구를 통해 배운 모든 것을 여러분과 공유하겠습니다.

이 글이 도움이 되셨기를 바라며, Svelte를 쉽게 시작하는 데 도움이 되기를 바랍니다.

제가 추천하는 메뉴는 다음과 같습니다:
- 핵심 개념 가이드
∘ 주요 기능
∘ 라이프 사이클
- 실용 가이드
∘ 프로젝트 구조
∘ 스타일 재사용
∘ 자식 호출 및 데이터 전달하기
∘ Http 요청
∘ 컴포넌트 상태
∘ 조건부 렌더링
∘ 목록
∘ 반응형 문
∘ 반응형 상태 "저장소"
∘ 탐색
∘ 전환 및 애니메이션
- 최적화 가이드
∘ 대용량 데이터 소스 처리하기
∘ 대규모 애플리케이션 처리하기
∘ 이미지 지연 로딩
∘ 로컬 파일 비동기 로드하기
- 에코시스템 가이드
