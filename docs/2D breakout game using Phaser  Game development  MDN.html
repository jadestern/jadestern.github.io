이 단계별 튜토리얼에서는 페이저 프레임워크를 사용하여 자바스크립트로 작성된 간단한 모바일 MDN 브레이크아웃 게임을 만듭니다.

모든 단계에는 편집 가능한 라이브 샘플이 제공되므로 중간 단계가 어떤 모습이어야 하는지 확인할 수 있습니다. 페이저 프레임워크를 사용하여 렌더링 및 움직이는 이미지, 충돌 감지, 제어 메커니즘, 프레임워크별 헬퍼 함수, 애니메이션 및 트윈, 승패 상태와 같은 기본적인 게임 메커니즘을 구현하는 기본 사항을 배우게 됩니다.

이 튜토리얼 시리즈를 최대한 활용하려면 이미 기초부터 중급 수준의 자바스크립트 지식이 있어야 합니다. 이 튜토리얼을 다 보고 나면 페이저로 간단한 웹 게임을 직접 만들 수 있을 것입니다.

학습 경로에 대한 참고 사항 - 순수 자바스크립트부터 시작하는 것이 웹 게임 개발에 대한 탄탄한 지식을 쌓는 가장 좋은 방법입니다. 아직 순수 자바스크립트 게임 개발에 익숙하지 않다면, 이 시리즈의 후속작인 순수 자바스크립트를 사용한 2D 브레이크아웃 게임을 먼저 제작해 보는 것이 좋습니다.

그 후에는 원하는 프레임워크를 선택하여 프로젝트에 사용할 수 있습니다. 저희는 지원과 커뮤니티가 잘 갖춰져 있고 플러그인 세트가 잘 갖춰진 탄탄한 프레임워크인 Phaser를 선택했습니다. 프레임워크는 개발 시간을 단축하고 지루한 부분을 처리하는 데 도움을 주므로 재미있는 부분에 집중할 수 있습니다. 하지만 프레임워크가 항상 완벽한 것은 아니므로 예상치 못한 일이 발생하거나 프레임워크가 제공하지 않는 기능을 작성하려면 순수한 자바스크립트 지식이 필요합니다.

참고: 이 글 시리즈는 실습용 게임 개발 워크샵의 자료로 사용할 수 있습니다. 페이저를 사용한 게임 개발에 대한 강연을 하고 싶다면 이 튜토리얼을 기반으로 한 Gamedev 페이저 콘텐츠 키트를 사용할 수도 있습니다.
